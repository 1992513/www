<details><summary align="center"><p align="center"><strong>历史的见证  :arrow_double_down: </strong></p></summary>

<p><strong>&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;二十年的腥风血雨，二十多年的凌辱折磨，岁月飞逝，历史见证，迫害善良原本就是违背道义不得民心的...</p>
