<h1><p><strong>еӮІйӣӘжў…иҠұ</strong></p></h1>
</div>

<details><summary align="center"><p align="center"><strong>еҺҶеҸІзҡ„и§ҒиҜҒ  :arrow_double_down: </strong></p></summary>

<p><strong>&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;дәҢеҚҒеӨҡе№ҙзҡ„и…ҘйЈҺиЎҖйӣЁпјҢдәҢеҚҒеӨҡе№ҙзҡ„еҮҢиҫұжҠҳзЈЁпјҢеІҒжңҲйЈһйҖқпјҢеҺҶеҸІи§ҒиҜҒпјҢиҝ«е®іе–„иүҜеҺҹжң¬е°ұжҳҜиҝқиғҢйҒ“д№үдёҚеҫ—ж°‘еҝғзҡ„...</p>
<p>&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;1999е№ҙдёӯе…ұеӣ жі•иҪ®еҠҹдҝ®зӮјдәәж•°и¶…иҝҮе…ҡе‘ҳиҖҢз–ҜзӢӮй•ҮеҺӢпјҢдёҚжғңеҲ¶йҖ дёҖдёӘдёӘи°ҺиЁҖпјҢеҜјжј”еӨ©е®үй—ЁиҮӘз„ҡжЎҲж Ҫиөғжі•иҪ®еҠҹпјҢеҜ№еқҡжҢҒдҝЎд»°зҡ„еӯҰе‘ҳз»‘жһ¶жҠ„е®¶пјҢз”Ёе°Ҫй…·еҲ‘пјҢеҜјиҮҙж•°д»ҘзҷҫдёҮи®Ўзҡ„е®¶з ҙдәәдәЎгҖҒеҰ»зҰ»еӯҗж•ЈпјҒз„¶иҖҢдёӯе…ұй«ҳдј°дәҶиҮӘе·ұзҡ„йҖ еҒҮе®Јдј гҖӮеҗ„еӣҪж”ҝеәңзә·зә·и°ҙиҙЈдёӯе…ұжҡҙиЎҢпјҢй«ҳеәҰиӮҜе®ҡжі•иҪ®еҠҹжҸҗеҚҮйҒ“еҫ·гҖҒеҮҖеҢ–еҝғзҒөгҖҒејәиә«еҒҘдҪ“зҡ„еҚ“и¶ҠжҲҗж•ҲгҖӮеңЁеӣҪеҶ…пјҢеЈ°жҸҙжі•иҪ®еҠҹзҡ„дәӢд»¶жӯӨиө·еҪјдјҸгҖӮжҲҗеҚғдёҠдёҮдәәдёәиў«йқһжі•жҠ“жҚ•зҡ„еӯҰе‘ҳиҜ·ж„ҝпјҢжј”з»ҺдәҶдёҚз•ҸејәжқғеҸҚиҝ«е®ізҡ„дёҖе№•еҸҲдёҖе№•гҖӮ</p>
<p>&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;дәҢеҚҒеӨҡе№ҙиҝҮеҺ»дәҶгҖӮеҰӮд»Ҡжі•иҪ®еҠҹвҖңзңҹе–„еҝҚвҖқжҷ®дё–д»·еҖји§ӮпјҢи¶…и¶Ҡж°‘ж—Ҹж–ҮеҢ–пјҢжҙӘдј дё–з•ҢдёҖзҷҫеӨҡдёӘеӣҪе®¶пјҢд»…еҸ°ж№ҫе°ұжңүе…«еҚҒдёҮдәәдҝ®зӮјпјҢе…¶дё»иҰҒи‘—дҪңиў«иҜ‘жҲҗеӣӣеҚҒеӨҡз§Қж–Үеӯ—пјҢдёҫдё–иөһиӘүпјҒиҖҢдёӯе…ұжұҹжіҪж°‘еҚҙеӣ зҪӘжҒ¶иҝ«е®іпјҢеңЁеӣҪеҶ…иў«дәҢеҚҒеӨҡдёҮдәәе®һеҗҚиө·иҜүпјҢеңЁжө·еӨ–иў«ж•°зҷҫдёҮдәәиҒ”зҪІдёҫжҠҘгҖӮеңЁиҝҷеңәеҜ№е–„иүҜжү“еҺӢдёӯпјҢдёӯе…ұжҠҠжі•еҲ¶гҖҒйҒ“еҫ·дјҰзҗҶйғҪжҺЁеҗ‘еҙ©жәғиҫ№зјҳгҖӮж°‘еҝғдё§е°ҪгҖҒеӨ©жҖ’дәәжҖЁ!дәәд»¬з»ҲдәҺи§үйҶ’пјҡиҰҒз»“жқҹиҝҷиҝ‘зҷҫе№ҙж°‘ж—Ҹжө©еҠ«пјҢйҮҚе»әзӨҫдјҡйҒ“еҫ·пјҢе”ӨйҶ’еӣҪдәәиүҜзҹҘпјҢе…Ёж°‘йҖҖеҮәдёӯе…ұжҳҜжӢҜж•‘дёӯеҚҺж°‘ж—Ҹзҡ„е”ҜдёҖеҮәи·ҜгҖӮ
<p>&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;дёӯе…ұжҙ»ж‘ҳжі•иҪ®еҠҹеӯҰе‘ҳеҷЁе®ҳиҙ©еҚ–зҡ„дәӢе®һжӣқе…үеҗҺйңҮжғҠдё–з•ҢгҖӮеӨ§зәӘе…ғзӨҫи®әгҖҠд№қиҜ„е…ұдә§е…ҡгҖӢд»Ҙж— еҸҜиҫ©й©ізҡ„дәӢе®һжҸӯзӨәдёӯе…ұиҝҷвҖңиҘҝжқҘе№ҪзҒөвҖқзҡ„зңҹйқўзӣ®пјҡе®ғеӯіз”ҹж— ж•°и…җиҙҘпјҢжҜҒеқҸдёӯеҚҺж–ҮжҳҺпјҢжҜҒзҒӯеӣҪдәәйҒ“еҫ·пјҢеңЁдёҖеңәеңәж”ҝжІ»иҝҗеҠЁдёӯеҗһеҷ¬ж•°еҚғдёҮеҗҢиғһз”ҹе‘ҪпјҢжҳҜзңҹе®һзҡ„йӯ”й¬јз»„з»ҮпјҢеҝ…йҒӯеӨ©жғ©!дёӯеӣҪдәәд»Һе°ҸеҠ е…Ҙе°‘е…Ҳйҳҹж—¶е°ұеңЁиЎҖж——дёӢе®ЈиӘ“иҰҒжҠҠз”ҹе‘ҪзҢ®з»ҷе®ғгҖӮеңЁжӯӨвҖңеӨ©зҒӯдёӯе…ұвҖқд№Ӣж—¶пјҢйҖҖе…ҡи§ЈйҷӨжҜ’иӘ“пјҢдёҚдёҺжҒ¶йӯ”дёәдјҚпјҢж—ўжҳҜйҒ“еҫ·иүҜзҹҘзҡ„йҖүжӢ©пјҢд№ҹжҳҜиҮӘжҲ‘дҝқе№іе®үзҡ„еӨ©ж„Ҹе®үжҺ’гҖӮеҰӮд»Ҡи¶…иҝҮ4дәҝдәәдёүйҖҖгҖӮе…Ёж°‘<a href="http://d2d1m4x4373e09.cloudfront.net/UmhjM" rel="nofollow">йҖҖеҮәдёӯе…ұ</a>пјҢи§ЈйҷӨиҮӘе·ұиә«дёҠзҡ„иҖ»иҫұжҺҲи®°пјҢж°ёиҝңз»“жқҹе…ұдә§е…ҡеҜ№дёӯеӣҪдәәзҡ„иҝ«е®іпјҢиҮӘеҲ©еҲ©д»–пјҢжҳҜжүҖжңүе–„иүҜдәәд»¬зҡ„ж„ҝжңӣпјҒ</strong></p>

<h1><p><strong>иҜ·еҸӮдёҺиҝҷеңәж°‘ж—ҸиҮӘж•‘иҝҗеҠЁ</strong></p></h1>
<p><strong>
</div>
<p>&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;2000е№ҙ10жңҲ30ж—ҘжҷЁпјҢйҮҚеәҶдә‘йҳідёүжҳҹжІұжұҹйқўпјҢж•°еҗҚеӣ вҖңй•ҝиҝҗ1еҸ·вҖқзҝ»иҰҶзҡ„иҗҪж°ҙиҖ…еңЁеҶ°еҶ·зҡ„жұҹж°ҙдёӯжҢЈжүҺгҖҒе‘јж•‘гҖӮе‘Ёеӣҙиҝ‘10жқЎе°Ҹжңәй©іиҲ№дё»д»¬еҜ№е‘јж•‘зҡ„дәәеҚҙи§ҶиҖҢдёҚи§ҒпјҢеҸӘйЎҫжү“жҚһж°ҙйқўзҡ„вҖңеҲЁиҙўвҖқпјҢеҮ дёӘиҲ№дё»иҝҳеҸ–з¬‘йҒ“пјҢдҪ д»¬е°ұеҘҪеҘҪеңЁиҲ№еә•дј‘жҒҜеҗ§гҖӮпјҲгҖҠйҮҚеәҶжҷҡжҠҘгҖӢпјү</p>
<p>&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;2003е№ҙ12жңҲ20ж—ҘеҮҢжҷЁпјҢзҸ жө·йҰҷжҙІвҖңж»ҡзҹівҖқзҡ„еЈ«й«ҳеӨ§еҺ…еҸ‘з”ҹдёҖиө·ејәеҘёжЎҲгҖӮ8еҗҚз”·еӯҗеҪ“зқҖеңЁеңәиҝ‘200дәәзҡ„йқўпјҢе°ҶдёҖеҘіеӯҗж‘ҒеңЁжІҷеҸ‘дёҠејәеҘёгҖӮпјҲгҖҠж–°дә¬жҠҘгҖӢпјү</p>
<p>&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;вҖңеҚ§й“әиҪҰејәеҘёжЎҲйңҮжғҠе№ҝиҘҝжӯ№еҫ’иҝһеҘё3еҘіж— дәәж•ўиЁҖвҖқ(гҖҠдёӯеӣҪж–°й—»зҪ‘гҖӢ2002е№ҙ07жңҲ05ж—Ҙ)</p>
<p>&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;вҖңе“Ҳе°”ж»ЁдёҠжј”йӘҮдәәдёҖе№•пјҡжӯ№еҫ’еҪ“иЎ—жқҖдәәж•°зҷҫеұ…ж°‘еӣҙи§ӮвҖқпјҲгҖҠдёӯеӣҪйқ’е№ҙжҠҘгҖӢ2002е№ҙ7жңҲ15ж—Ҙпјү</p>
<p>&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;вҖңе°‘еҘіжӢ’и°ғжҲҸиў«з Қе…«еҲҖдәҢеҚҒеӨҡдёӘеӣҙи§ӮиҖ…ж— еҠЁдәҺиЎ·вҖқпјҲгҖҠеҚ—ж–№зҪ‘гҖӢ2002е№ҙ5жңҲ17ж—Ҙпјү</p>
<p>&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;вҖҰвҖҰ</p>
<p>&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;иҝҷд»ӨдәәдёҚеҜ’иҖҢж —зҡ„жё…еҚ•пјҢиҝҳеҸҜд»ҘдёҖзӣҙеҲ—дёӢеҺ»гҖӮдёҖдёӘжӣҫз»ҸеҲӣйҖ иҫүз…Ңж–ҮжҳҺзҡ„зӨјд»Әд№ӢйӮҰпјҢдҪ•д»Ҙжј”еҸҳеҲ°иҝҷзӯүеҶ·жј жҲҗжҖ§зҡ„еҸҜжӮІең°жӯҘгҖӮж•°еҚғе№ҙзҡ„йҮҚеҫ·иЎҢе–„гҖҒвҖңд»Ғд№үзӨјжҷәдҝЎвҖқзҡ„зҫҺеҘҪд»·еҖје“ӘйҮҢеҺ»дәҶ?</p>
<p><img ALIGN=left src="https://raw.githubusercontent.com/1992513/www/master/t/2224dae6.jpg">&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;дёӯе…ұзӘғеӣҪеҗҺпјҢе°Ҫз®ЎжҲ‘д»¬зҡ„иЎҖи„үд»ҚеңЁе»¶з»ӯпјҢдҪҶжҳҜеҚҺеӨҸж–ҮжҳҺзҡ„и–ӘзҒ«жүҝдј е·Із»Ҹиў«жҲӘж–ӯгҖӮдј з»ҹзҡ„дҝЎд»°е’Ңд»·еҖји§Ӯиў«ејәеҠӣз ҙеқҸпјҢдәәдёҺдәәд№Ӣй—ҙзҡ„е…ізҲұдёҺе’Ңи°җиў«жүӯжӣІжҲҗдәҶж–—дәүдёҺд»ҮжҒЁпјӣдёҖеҲҮе’ҢеҺҶеҸІдј з»ҹжңүе…ізҡ„зү©е“Ғиў«еҪ“жҲҗвҖңеӣӣж—§вҖқпјҢеҮ еҚғе№ҙжқҘвҖңеҝ еӯқиҠӮд№үвҖқвҖңеӨ§еӯҰд№ӢйҒ“пјҢеңЁжҳҺжҳҺеҫ·пјҢеңЁдәІж°‘пјҢеңЁжӯўдәҺиҮіе–„вҖқзҡ„жҷ®дё–д»·еҖји§Ӯиў«еҪ“жҲҗе°Ғе»әзіҹзІ•гҖӮ"дҝ®иә«йҪҗе®¶жІ»еӣҪе№іеӨ©дёӢ" "дёҠиҮіеӨ©еӯҗдёӢиҮіеә¶ж°‘пјҢдёҖзҡҶд»Ҙдҝ®иә«дёәжң¬"зҡ„дё»жөҒж–ҮеҢ–пјҢиў«дјӘеҢ–жҲҗдәҶе®«еәӯдәүж–—зӯүе…ҡж–ҮеҢ–гҖӮз”ұжӯӨеёҰжқҘзӨҫдјҡйҒ“еҫ·е…Ёйқўеҙ©жәғпјҢдҪҝж•ҙдёӘж°‘ж—Ҹйҷ·е…Ҙж·ұйҮҚзҡ„еҚұжңәгҖӮ</p>
<p>&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;жҲ‘д»¬жҲҗдәҶдёҖдёӘеӨұеҺ»дәҶиҮӘжҲ‘зҡ„ж°‘ж—ҸгҖӮд»Һж–ҮеҢ–дёҠзңӢпјҢжҲ‘д»¬е·Із»ҸеҸҳжҲҗдәЎеӣҪеҘҙгҖӮ</p>
<p>&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;дёӯеҚҺж°‘ж—Ҹдә”еҚғе№ҙжј«жј«еҺҶеҸІй•ҝжІідёӯпјҢжӣҫз»Ҹд№ҹеҮәзҺ°иҝҮжңқд»ЈжӣҙеҸ пјҢд№ҹеҮәзҺ°иҝҮжЎҖгҖҒзәЈдёҖзұ»зҡ„жҡҙеҗӣж°‘иҙјпјҢз„¶иҖҢж— и®әеӣҪе®¶еҰӮдҪ•е…өиҝһзҘёз»“пјҢй»Һж°‘еҰӮдҪ•ж°ҙж·ұзҒ«зғӯпјҢжұҹеұұжҳ“дё»д№ӢеҗҺгҖӮеҸӘйңҖеҮ еҚҒе№ҙдј‘е…»з”ҹжҒҜпјҢж–ҮжҳҺдҫҝйҮҚж–°жҳҢжҳҺе…үеӨ§гҖӮжӯӨзҡҶеӣ ж°‘ж—Ҹж–ҮеҢ–дҫқж—§пјҢж°‘ж—ҸйҒ“еҫ·д»ҚеңЁпјҢж°‘ж—ҸзІҫзҘһжңӘжӯ»пјҒ</p>
<p>&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;еҺҶеҸІиҫ—иҪ¬еҲ°д»Ҡж—ҘпјҢд»Ҙи°ҺиЁҖе’ҢжҡҙеҠӣеҫ—йҖһзҡ„дёӯе…ұзӘғжқғеҗҺпјҢдёӯеҚҺж°‘ж—Ҹз”ҹеӯҳж №еҹәиў«ж‘§жҜҒеҫ—еҚғз–®зҷҫеӯ”пјҡдә”еҚғе№ҙдј жүҝзҡ„йҒ“еҫ·ж ҮеҮҶиў«жӯӘжӣІејӮеҢ–пјӣдҝЎд»°еҮәзҺ°еҚұжңәпјҢйҖ жҲҗдёӯеӣҪиҙӘе®ҳйҒҚең°пјҢзү©ж¬ІжЁӘжөҒпјҢй»‘зӨҫдјҡж— еӯ”дёҚе…ҘгҖӮеқ‘и’ҷжӢҗйӘ—ж— еӨ„дёҚеңЁгҖҒеҒҮеҶ’дјӘеҠЈдә§е“Ғжіӣж»ҘжҲҗзҒҫгҖӮеҺҶе°ҪжІ§жө·жЎ‘з”°жүҚеҫ—д»ҘеҪўжҲҗзҡ„дёҚеҸҜеҶҚз”ҹзҡ„иҮӘз„¶иө„жәҗиў«зіҹи№ӢжұЎжҹ“гҖӮеҰӮжӯӨиҖҢеҫҖпјҢж— йңҖеӨҡж—ҘпјҢеӯҗеӯҷеҗҺд»Је°Ҷж— еӨ„еҜ»и§…дёҖж–№еҮҖеңҹпјҢдёҖжҺ¬еҮҖж°ҙпјҢдёҖеҸЈзәҜеҮҖзҡ„з©әж°”гҖӮ</p>
<p>&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;дәҺдёӯе…ұж— жүҖдёҚеңЁзҡ„ж¬әйӘ—жҙ—и„‘дёҺжҡҙеҠӣиғҒиҝ«дёӢпјҢжңүеӨҡе°‘дәәж”ҫејғдәҶиҜҡе®һе®ҪеҺҡгҖҒд»ҒзҲұи°Ұе’Ңзҡ„йҒ“еҫ·еә•зәҝпјҢеҸҳеҫ—и§ҒйЈҺдҪҝиҲөгҖҒиҗҪдә•дёӢзҹіпјҢз”ҡиҮідёәдёҖе·ұд№Ӣз§ҒиҖҢж— жҒ¶дёҚдҪңгҖӮ</p>
<p>&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;зЁҚеҫ®зӣҳзӮ№дёӯе…ұзҡ„зҪӘжҒ¶пјҢи®©дәәж„ҹеҲ°еҝғжғ…ж— жҜ”зҡ„жІүйҮҚгҖӮдәҶи§ЈдәҶдёӯе…ұпјҢжүҚжҳҺзҷҪд»Җд№ҲеҸ«еҒҡзҪӘжҒ¶ж»”еӨ©пјҢзҪ„з«№йҡҫд№ҰгҖӮ</p>
<p>&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;&nbsp;&#160;еҺҶеҸІдёҠж— ж•°жӣҫдёҚеҸҜдёҖдё–зҡ„еёқеӣҪйҡҸзқҖйҒ“еҫ·е •иҗҪе’Ңзү©иҙЁиҚ’ж·«иҖҢеҪ’дәҺе°ҳеңҹпјҢдёӯе…ұжӯЈиёҸеңЁиҝҷдёҖиҪҰиҫҷдёӯпјҢдҪҶжҳҜе®ғеҜ№еҚҺеӨҸеӯҗж°‘жҡҙеҠӣжҙ—и„‘пјҢзҒҢиҫ“дәҶе®ғзҡ„е…ҡж–ҮеҢ–пјҢжҚҶз»‘ж•ҙдёӘж°‘ж—ҸпјҢжӢ–еҗ‘дёҮеҠ«дёҚеӨҚзҡ„ж·ұжёҠгҖӮжҖҺж ·еҪ»еә•зҡ„жҠҠеҚҺеӨҸд»Һз”ҹжӯ»дёҖзәҝд№ӢеҚұжңәдёӯи§Јж•‘еҮәжқҘпјҹиҜ·дҪ еҠ е…ҘеҲ°иҝҷеңәиҪ°иҪ°зғҲзғҲзҡ„ж°‘ж—ҸиҮӘж•‘иҝҗеҠЁдёӯжқҘ-----<a href="http://d2d1m4x4373e09.cloudfront.net/UmhjM" rel="nofollow">еЈ°жҳҺйҖҖеҮәе…ұдә§е…ҡгҖҒе…ұйқ’еӣўгҖҒе°‘е…Ҳйҳҹз»„з»Ү</a>
